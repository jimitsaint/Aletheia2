# Aletheia — Jimi T Saint (React + Vite + React Router)

Färdigt för Netlify.

## Bygg i Netlify
- Build command: `npm run build`
- Publish directory: `dist`

## Struktur
- `src/App.jsx` — Meny + routes
- `src/pages/*` — En sida per menyval
- `public/_redirects` — SPA fallback (viktig för direkta länkar)
